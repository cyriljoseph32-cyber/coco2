## 2) Domaine + Google Search Console  *(créneau jeudi 18/06)*

### a) Domaine (≈ 1 h, gros impact crédibilité + SEO)
1. Achète un domaine — suggestions : **cocosamui.ai**, cococoncierge.com, asksamui.com, cocokohsamui.com.
   (Namecheap / Cloudflare / Porkbun, ~10–40 €/an.)
2. Vercel → ton projet → **Settings → Domains → Add** → colle le domaine → suis les enregistrements DNS indiqués.
3. Quand le domaine est actif, dis-le-moi : je remplace l'ancien lien `vercel.app` partout (site, 34 brouillons, QR, schema `url`/`sameAs`).

### b) Google Search Console (≈ 15 min)
1. **search.google.com/search-console** → Ajouter une propriété → préfixe d'URL = ton site.
2. Vérification recommandée : **balise HTML** → Google te donne une balise `<meta name="google-site-verification" ...>`. Envoie-la-moi, je l'ajoute dans `public/index.html` (head), tu redéploies, tu cliques Vérifier.
3. Une fois vérifié → **Sitemaps** → soumets `sitemap.xml`. (Déjà en ligne : `/sitemap.xml` et `/robots.txt`, avec les bots IA autorisés ✅.)
4. Demande l'indexation de la page d'accueil (« Inspection de l'URL » → « Demander l'indexation »).
